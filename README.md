# Snake Game

A classic **Snake Game** built with **HTML, CSS, and jQuery**. Control the snake using arrow keys, eat the red food to grow, and avoid running into walls or yourself. The game keeps track of your score and works in any modern web browser.

## Features
- Classic snake gameplay
- Score counter
- 20x20 grid game board
- Game over alert on collision
- Easy to customize (grid size, speed, colors)

## How to Play
1. Open `index.html` in your browser.
2. Use **arrow keys** to move the snake.
3. Eat the red food to increase your score and grow the snake.
4. Avoid hitting walls or yourself.

## Technologies Used
- HTML
- CSS (Grid Layout)
- jQuery (DOM manipulation & game logic)
